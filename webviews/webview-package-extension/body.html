Publisher ID

Your Visual Studio Marketplace Publisher ID. Manage your publishers and extensions.

Extension Display Name

Display name of the extension. Must be unique in the Visual Studio Marketplace. Shown in the Visual Studio Marketplace.

Extension Name

Name of the extension. Must be unique in the Visual Studio Marketplace. Not displayed on the UI.

Extension Version

Version of the extension. Must follow semantic versioning.

Extension Description

Short description of what the extension is and does. Shown in the Visual Studio Marketplace.
